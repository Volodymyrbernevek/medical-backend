# Medical Platform Backend 🚀

https://medical-backend-api-n77h.onrender.com/docs#/
Бекенд-частина веб-платформи для медичного обліку, ведення карток хвороб та безпечного обміну медичними документами між пацієнтами та лікарями.

## 🛠 Технологічний стек
* **Framework:** FastAPI (Python 3.11)
* **Database:** PostgreSQL (Neon.tech Cloud)
* **ORM:** SQLAlchemy
* **Linter/Formatter:** Ruff
* **Containerization:** Docker
* **Cloud Storage:** Cloudinary (для завантаження медичних PDF та зображень)
* **Hosting/Deployment:** Render (Free tier)
